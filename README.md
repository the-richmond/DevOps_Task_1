# Array Processor (Variant 19)
[![CI/CD Pipeline](https://github.com/the-richmond/DevOps_Task_1/actions/workflows/ci.yml/badge.svg)](https://github.com/the-richmond/DevOps_Task_1/actions)
**Версия:** 1.0.0

## Описание проекта
Данный проект разработан в рамках курса "Управление программными проектами" (DevOps). 
Программа на C++ обрабатывает два массива `X(10)` и `Y(10)` и формирует новый массив `Z` по следующим правилам:
* Включает элементы `X`, которые больше максимального элемента `Y`.
* Включает элементы `Y`, которые меньше минимального элемента `X`.
Программа включает защиту от некорректного пользовательского ввода.

## Структура репозитория
* `src/` — Исходный код программы (`main.cpp`) и `Makefile` для сборки.
* `cicd/` — Unit-тесты (`test_logic.cpp`, `test_errors.cpp`, `test_empty.cpp`) для автоматического тестирования.
* `DEBIAN/` — Конфигурационный файл `control` для сборки `.deb` пакета.
* `.github/workflows/` — Конфигурация CI/CD пайплайна (`ci.yml`).

## CI/CD Pipeline
Проект использует GitHub Actions для непрерывной интеграции. При каждом `push` или `pull_request` в ветку `main` автоматически выполняются:
1. Сборка бинарного файла (`make`).
2. Прогон Unit-тестов.
3. Сборка релизного `.deb` пакета.
